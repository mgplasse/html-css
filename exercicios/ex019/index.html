Semântica Web

Um <ID> vai identificar um elemento único dentro da página atual. Essa identificação vai nos permitir criar um estilo especial para um elemento isolado.

Já um <CLASS> vai identificar uma classe à qual um ou mais elementos pertençam, compartilhando características em comum a todos que façam parte deste grupo (Falando em CSS)


SE LIGA NA REGRA: Em um mesmo doc HTML, só podemos usar um único ID, isso significa que não podemos ter dois ID's dentro da mesma página. Porém, podemos atribuir um mesmo class para vários elementos que possuam essa mesma característica.
